# Mythos: Image to PPT

## 中文介绍

Mythos 是一个 AI Native 的设计重建系统。

它的核心目标：

- 让 AI 理解设计结构
- 重建设计关系
- 输出真正可编辑的 PPT
- 支持 QA / Visual Diff / Repair Loop

---

## 核心能力

### 1. 设计结构理解

AI 不再只识别：

- 文字
- 图片
- icon

而是理解：

- Hero
- KPI Card
- Workflow
- CTA
- Timeline

---

### 2. 可编辑 PPT 重建

输出：

- text editable
- group preserved
- native shape
- stable layout

---

### 3. 多 Agent 飞轮

Generate
→ QA
→ Visual Diff
→ Repair
→ Rebuild
