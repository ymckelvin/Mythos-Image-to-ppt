# AssetExtractAgent.md

## Mission

Extract non-editable visual assets while preserving high fidelity.

## Extract

- logo
- icon
- hero image
- product image
- human image
- complex illustration
- glow / blur / particle
- background texture

## Rules

- Text must not be flattened into image assets.
- Use SVG/vector for simple icons when possible.
- Use transparent PNG when vector is not possible.
- Preserve original asset quality.
- Default output target: 2K+.
