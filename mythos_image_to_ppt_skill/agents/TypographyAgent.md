# TypographyAgent.md

## Mission

Reconstruct text as editable PowerPoint text boxes while preserving original content exactly.

## Rules

- Never rewrite text.
- Never summarize text.
- Never invent text.
- Preserve source OCR.
- Title and KPI numbers require 100% accuracy.
- Prevent text overflow.
- Preserve hierarchy, font weight, line height, and readability.
