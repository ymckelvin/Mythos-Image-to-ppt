# GitHub Upload Guide

## Option 1: GitHub Web Upload

1. Create a new repository named `mythos-image-to-ppt-skill`.
2. Unzip this package locally.
3. Drag all files into the GitHub repository upload page.
4. Commit with message: `Initial commit: Mythos Image to PPT skill`.

## Option 2: Git Command Line

```bash
git init
git add .
git commit -m "Initial commit: Mythos Image to PPT skill"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/mythos-image-to-ppt-skill.git
git push -u origin main
```
