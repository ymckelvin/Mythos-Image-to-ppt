# Quality Rubric

## Priority 1 — Editable Fidelity

- Text must be editable.
- Shapes should be native where possible.
- Cards, buttons, workflow nodes must preserve group structure.
- Full-slide rasterization is forbidden.

## Priority 2 — Text Fidelity

- No wrong characters.
- No missing text.
- No garbled text.
- No AI rewriting.
- Titles and KPI numbers must be 100% accurate.

## Priority 3 — Visual Fidelity

- Source visual hierarchy must be preserved.
- Main visual must match source.
- Layout and density must match source.
- Assets must be high-resolution, default 2K+.

## Priority 4 — Layout Stability

- Alignment must be stable.
- Spacing must be consistent.
- Reading order must be preserved.
